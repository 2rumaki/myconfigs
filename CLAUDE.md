# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

PCを買い替えた際などに開発環境の復旧を容易かつ確実に実行できるようにすることを目的とする。dotfile などの各種設定ファイルとそのセットアップ手順を管理している。現在は README.md にツールのインストール手順・設定方法をまとめている。

## ワークフロー

- **Issue**: 導入したいツールやサービスを Issue として登録する
- **PR**: Issue に対応する PR で、ツールの導入手順や設定をリポジトリに取り込む

- `README.md` - ターミナル環境のセットアップガイド（zsh, Starship, HackGen, neovim/LazyVim, Catppuccin, Lazygit, delta, Atuin）
